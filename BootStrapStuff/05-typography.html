Typography

Bootstrap's global default font-size is 14px, with a line-height of 1.428. The default fonts used by Bootstrap are "Helvetica Neue", Helvetica, Arial, sans-serif;. These properties are applied to the body of your document along with all p tags. In addition, all p tags have a margin-bottom applied to them that is equal to 1/2 of their line height (10px). You can of course override any of these behaviors using your own CSS rules.

Bootstrap has a built in color palette - you can reference this palette using the following classes:

.text-muted

.text-primary

.text-success

.text-info

.text-warning

.text-danger

We can also use a set of similar classes to style the background color of an element, like this:
